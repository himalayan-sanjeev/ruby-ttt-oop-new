### Tic-Tac-Toe, a cli game built in Ruby (OOP approach)


### Created with
* Ruby : 3.0.2

### Test
* rspec : 3.10

### Code Formating
* rubocop : 1.24.1

## To run this project
```
 $ git clone https://github.com/himalayan-sanjeev/ruby-ttt-oop-new
 $ cd ruby-ttt-oop-new
 $ cd bin
 $ ruby main.rb
```
